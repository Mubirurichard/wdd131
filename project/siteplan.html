Website Site Plan

WDD 131 Final Project Planning Document

Site Name

Explore Uganda

The name was chosen to represent a tourism-focused website that highlights Uganda’s natural beauty, culture, and travel destinations. It is simple, memorable, and clearly communicates the site’s purpose.

Optional domain: explore-uganda.org

Site Purpose

The purpose of this website is to provide travelers and locals with information about popular destinations in Uganda, including national parks, cultural sites, and travel tips. The site will also include an inquiry form and interactive features to enhance user engagement.

Scenarios

  • Which national parks in Uganda are best for wildlife viewing?
  • How can I contact local tour guides or get travel recommendations?

Color Schema

The following colors were selected to reflect nature and readability:

  • Forest Green (#2e7d32): Used for headings and navigation
  • Light Beige (#f4f4f4): Used for page background
  • Dark Gray (#333333): Used for body text

Typography

  • Montserrat: Used for headings
  • Roboto: Used for body text and paragraphs
